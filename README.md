# ANLP_Project
Repository on ANLP code for KG Project

hop_prediction_bert_linear.py : represents all the notebooks used to train hop prediction model with linear classification head
hop_prediction_bert_mlp.py : represents all the notebooks used to train hop prediction model with MLP Classification head
hop_prediction_bert_regression.py : represents all the notebooks used to train hop prediction model with regression head
knowledge_adaptor.ipynb : Jupyter notebook used to train, test the knowledge adaptor
neo4j_creation_dataset_builder.ipynb: Jupyter notebook used to create neo4j database, work on retrieval and ranking from WebQSP dataset
preprocess_networkx.py: Attempt to create networkx graph from freebase on ada
retrievel_methods_exploration.ipynb: File Base Search, SQLite, NetworkX code for creation and retrievel
simple_inference_with_LLM.ipynb: Code for Direct Inference using SLM
